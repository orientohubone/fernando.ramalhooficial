# Checklist Operacional — Arquitetura Cognitiva de Categoria™

---

## FASE 1 — DESCOMPRESSÃO COGNITIVA

*(Quebra da categoria atual)*

### Objetivo

Revelar e enfraquecer o campo mental que aprisiona a marca.

### Checklist

* [ ] Mapear como o mercado define o “problema”
* [ ] Listar palavras repetidas da categoria
* [ ] Identificar promessas genéricas
* [ ] Mapear onde todas as marcas “soam iguais”
* [ ] Identificar gatilhos de desconfiança
* [ ] Identificar medo dominante do cliente
* [ ] Mapear objeções recorrentes
* [ ] Detectar onde a decisão trava

---

## FASE 2 — ARQUITETURA DE TERRITÓRIO

*(Reescrita perceptiva)*

### Objetivo

Criar um novo campo mental exclusivo.

### Checklist

* [ ] Redefinir o problema central da categoria
* [ ] Criar novo eixo de comparação
* [ ] Declarar o “jeito certo” de fazer
* [ ] Declarar o que passa a ser ultrapassado
* [ ] Criar nome para o novo território
* [ ] Criar tese central repetível
* [ ] Definir símbolos verbais da nova categoria
* [ ] Criar metáforas proprietárias

---

## FASE 3 — SEGURANÇA COGNITIVA

*(Confiança implícita)*

### Objetivo

Fazer o cérebro sentir que o novo campo é seguro.

### Checklist

* [ ] Simplificar narrativa ao essencial
* [ ] Reduzir opções
* [ ] Definir arquitetura de escolha
* [ ] Criar pontos de familiaridade
* [ ] Criar símbolos de autoridade
* [ ] Construir prova social estratégica
* [ ] Criar repetição perceptiva
* [ ] Eliminar ambiguidade

---

## FASE 4 — ATIVAÇÃO SILENCIOSA

*(Instalação mental no mercado)*

### Objetivo

Fazer o mercado internalizar o novo campo.

### Checklist

* [ ] Publicar micro-rupturas cognitivas
* [ ] Repetir a tese central
* [ ] Repetir símbolos proprietários
* [ ] Criar posts de reorganização mental
* [ ] Criar conteúdo de fricção cognitiva
* [ ] Neutralizar linguagem concorrente
* [ ] Desvalorizar mentalmente o modelo antigo
* [ ] Tornar seu modelo “o normal”

---

## FASE 5 — DOMÍNIO DE CATEGORIA

*(Referência implícita)*

### Objetivo

Virar o padrão mental da categoria.

### Checklist

* [ ] Criar linguagem própria recorrente
* [ ] Ver pessoas repetindo seus termos
* [ ] Receber convites por “visão”
* [ ] Ser citado sem estar presente
* [ ] Perder sensibilidade a preço
* [ ] Ser percebido como referência
* [ ] Gerar demanda sem venda ativa
* [ ] Tornar-se “o jeito certo” do mercado

---

## Regra Operacional

> Quando sua linguagem começa a circular sem você,
> o campo mental já foi tomado.
